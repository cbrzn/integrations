# IPFS Web3API package

Inspect schema files to see what functionalities are available - `/query/schema.graphql` and `/mutation/schema.graphql`.

## How To Run

### 1. Setup Test Env

```
yarn test:env:up
```

### 2. Build & Deploy The Web3API

```
yarn deploy
```

### 3. Test The Web3API Using A Query Recipe

```
yarn test
```

#### Remove Test Env

```
yarn test:env:down
```
